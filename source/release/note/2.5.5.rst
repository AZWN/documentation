=============
Spoofax 2.5.5
=============

Spoofax 2.5.5 is a minor bugfix release.

Changes
-------

Overall
~~~~~~~

NaBL2/Statix
~~~~~~~~~~~~

- Fix bug that threw away error messages in unchanged files if other files changed.
